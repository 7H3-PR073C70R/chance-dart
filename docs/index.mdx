# Chance Dart

## About

Chance Dart is a free Open Source project that allows you to generate random strings,
numbers, etc. to help reduce some tediousness particularly while writing
automated tests or anywhere else you need anything random.

Data generation is an important part of many projects especially projects that requires code generation and mocking of data.

Chance Dart is developed to deliver instant values to ensure speediness in development process and also reduce monotony.

Features includes:

- Instant value generation directly from defined methods.
- CLI to help generate random values outside the code editor.
- Files and value generation using build_runner alongside defined annotations.

## Should I use Chance Dart?

Chance Dart is a simple and fast way to generate a random values with little to no effort. It generally works for every Framework that uses Dart language.
